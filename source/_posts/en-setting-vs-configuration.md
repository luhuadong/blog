Setting 一般指设置，Configuration 一般指配置。两者的意思很接近，Setting 更偏向于给用户或运维人员使用的设定，而 Configuration 更偏向于给开发人员使用的设定，例如软件开发中用来避免一些 Hard Code 的可参数化的设定。
