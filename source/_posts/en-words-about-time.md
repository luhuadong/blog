## 星期（Days of the Week）

| 单词      | 缩写 | 中文   |
| --------- | ---- | ------ |
| Monday    | Mon. | 星期一 |
| Tuesday   | Tue. | 星期二 |
| Wednesday | Wed. | 星期三 |
| Thursday  | Thu. | 星期四 |
| Friday    | Fri. | 星期五 |
| Saturday  | Sat. | 星期六 |
| Sunday    | Sun. | 星期日 |

## 季节（Seasons）

| 季节          | 缩写       | 中文 |
| ------------- | ---------- | ---- |
| Spring        | SS（春夏） | 春季 |
| Summer        | SS（春夏） | 夏季 |
| Winter        | FW（秋冬） | 秋季 |
| Fall / Autumn | FW（秋冬） | 冬季 |

| -------------- | ------ |
| in the morning | 在早上 |
| last night     | 昨晚   |
| next year      | 明年   |
| today          | 今日   |
| at night       | 晚上   |
