如何克服拖延症？

发表于 2022-04-28 本文字数 826 字 阅读时长 1 分钟
  • Freedom Blog
首页 人生管理术

你认为引起拖延的原因是什么？据不完全统计，有 21% 的人认为引起拖延的原因是因为 “完美主义”，23% 的人则认为是因为不自信，26% 的人认为时间充足、不着急做。另外有 42% 的人纯粹是因为想偷懒，能拖就拖。你有没有中枪？

那要怎么样才能打败拖延症，提高我们的工作效率，让生活变得越来越惬意呢？下面介绍几种方法 👇

# 1、立即执行

研究认为，开始着手一件事是最大的障碍！幸运的是，人们天生有一种 “万事有始有终” 的驱动力，它会迫使人们完成已经开始了的工作。当我们没有完成任务时，就会经历不适和入侵性的想法。

# 2、时间管理

根据时间管理四象限法则，将待办事项按照重要程度排序，为自己设置时间限制，把它写下了记在日历上。这样，完成任务的可能性就更大！在此之上，还可以建立一个日程表来记录各项工作的进度，一栏写上时间跨度，另一栏写上这段时间内所完成的项目。记录进度可以实事求是地评估自己的工作，从而不会骄傲地认为自己已经完成了任务。

# 3、摆脱分心

拒绝网聊、关掉微博、不打开网页… 即使是邮件也在固定时间段处理，甚至可以断掉网络。

最佳收发邮件时间：

  • 上班后的两小时
  • 下午上班前半小时
  • 下午三四点左右
  • 下班前

# 4、建立奖励机制

每次按时完成工作时给自己一个奖励，强化完成任务的成就感。下次想要拖延时，这份成就感就会自己冒出来鼓励你开始行动！

# 5、化整为零

高难度的任务让人心生焦虑、望而却步，解决办法就是将其分成一个个小任务，看到实现的可能性，大脑就会降低对其困难度的评估，再着手行动就会轻松许多。

# 6、停止多任务

研究认为，并行多任务时效率反而更低，因为多任务的频繁切换意味着注意力的不断转移。每一次转移，大脑都需要时间重新投入。倒不如把所要做的事列一张清单，这样更容易贯彻你的行动计划，每一次只专注做一件事，你的工作效率就会提高。

# 7、感兴趣的工作留到最后

将自己感兴趣的工作留到最后！为了最后能做自己喜欢的事情，前面做些不喜欢的工作也不是那么难以忍受嘛！

延迟满足，就是为了更有价值的长远结果！
