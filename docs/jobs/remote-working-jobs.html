寻找远程工作的网站平台

发表于 2023-01-24 更新于 2023-03-25 分类于 新时代新工作

GitHub

  • remote-jobs-in-china：支持远程办公的中国公司
  • awesome-indie-zh：独立开发/自由职业/远程工作资源列表
  • remote-working：收集整理远程工作相关的资料
  • awesome-freelancer：自由职业者&远程工作者&独立开发者&开源工作者@生存指南
  • remote-weekly：中国远程工作者周刊

电鸭社区

国内最活跃的远程工作者社区，这里有许多国内远程团队的分享和职位更新。如果你想了解国内远程工作的情况，可以在论坛上认识一下。

链接：https://eleduck.com

远程客

远程客是一个关注远程工作发展的网站，会定时更新远程工作的文章和工作机会，收集远程工作相关的工具。

链接：https://yuanchengke.com

Upwork

Upwork 是全球最大的、最优秀的、最规范的综合类人力外包服务平台，由著名的 Elance 和 oDesk 合并。这里聚集 900 万来自全球各地的自由工作者，你肯定可以在找到适合你的职位。

链接：https://www.upwork.com

Freelancer

Freelancer 的工作类型覆盖了很多不同的领域，由程序开发到市场营销、广告、会计、法务等一系列的可以远程的工作。

链接：https://www.freelance.com

Dribbble

你没看错，Dribbble 不只是全球最受欢迎的设计师社区，同样是设计师寻找远程工作的好出处。自从被 Tiny 收购后，Dribbble 的招聘属性正在慢慢增强，试着持续 PO 出自己的好作品，等待你的伯乐，同样你可以关注 Jobs 页面，给心仪的 Team 提交简历。

链接：https://dribbble.com/jobs

Stackoverflow（已关闭）

被广大程序员熟知的 Stackoverflow 本身就是一个全民远程工作的公司，程序员在远程工作中有很大的先天优势，Stackoverflow Jobs 里有远程职位的筛选，这里有丰富的远程技术职位。

链接：https://stackoverflow.com/jobs

99designs

99designs 是一个专门针对设计师的雇佣平台，你可以在这里参加设计竞赛给公开的项目投标，也可以给雇主提供一对一的专业服务。

链接：https://99designs.com.sg

Remoteok

Remoteok 不仅提供最初的兼职类远程工作，还有全职类，签署合同类和实习类的工作。网站创始人 Pieter Levels 本身就是一名数字游民，他同样是 Nomadlist 的创始人。

链接：https://remoteok.com

Toptal

Toptal 是一个高端一些的自由职业者平台，适合比较有经验和工作尽力的远程工作者。它将企业与全球的软件工程师，设计师和业务顾问联系起来。

链接：https://www.toptal.com

Topcoder

Topcoder 通过算法比赛吸引世界顶级的程序员，他会将一下大型项目分割成很多小模块，通过竞赛的模式交给用户来做，优胜者可以拿到制定模块的奖金。

链接：https://www.topcoder.com
