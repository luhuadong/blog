将 Markdown 格式文章发布到微信公众号

发表于 2023-01-27 本文字数 307 字 阅读时长 1 分钟

首页 一起学Markdown

如果你和我一样，平时习惯使用 Markdown 写作，当我们想把文章发布到微信公众号或知乎等平台时，就会发现挺麻烦的，尤其是排版方面，因为它们并不支持 Markdown 格式。

那有没有简单高效的办法呢？答案是有的！

你可以借助微信公众号 Markdown 在线编辑器，将写好的 Markdown 文章粘贴过去，调整排版主题，即可生成适合微信公众号或知乎风格的文章。

推荐使用下面几款工具：

  • https://markdown.com.cn/editor/
  • https://wangchujiang.com/wxmp/
  • https://lab.lyric.im/wxformat/
  • http://md.aclickall.com/
