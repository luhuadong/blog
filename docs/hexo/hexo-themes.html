首页 如何搭建Hexo博客

Hexo 博客系统的主题有很多选择，你可以在 这里 找到它们。如果你愿意，你还可以自己动手编写一个属于你的主题。

本文主要介绍几个比较好用的主题。

# NexT

NexT 主题的特点是简洁，提供三种外观风格，但同时也功能齐全，而且插件丰富，因此是 Hexo 主题中非常流行的一个。

  • GitHub：https://github.com/next-theme/hexo-theme-next
  • 项目官网：https://theme-next.iissnan.com

# Fluid

Fluid 一款 Material Design 风格的 Hexo 主题，由 Fluid-dev 负责开发与维护。

  • GitHub：https://github.com/fluid-dev/hexo-theme-fluid
  • 项目官网：https://hexo.fluid-dev.com

# Butterfly

Butterfly 是 Jerry 在 hexo-theme-melody 主题的基础上进行开发，支持多种美化魔改、多功能组件及静态部署集成的 Hexo 主题。

  • GitHub：https://github.com/jerryc127/hexo-theme-butterfly
  • 项目官网：https://butterfly.js.org
