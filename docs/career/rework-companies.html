# 国内

# 极狐 GitLab

极狐 GitLab，是一应俱全的一站式 DevOps 平台，提供与独立应用完全一致的使用体验，从根本上改变开发、安全和运维团队的协同工作方式。极狐 GitLab 专注于解决中国用户需求的，是 GitLab DevOps 平台的中国发行版。

极狐（GitLab）公司以 “开源开放，人人贡献” 为原则，致力于推进开源生态与产业的结合。我们希望通过极狐 GitLab 开源平台，为中国用户提供全球领先的 “软件创新加速器”。

网址：https://gitlab.cn

# 国外

# FOSSASIA

FOSSASIA 是帮助自由和开源技术开发者和 Maker 的非营利组织。它由 Hong Phuc Dang 和 Mario Behling 成立。FOSSASIA 的目的是发展和套用开源技术以配合社会转变，并把重点放在亚洲用户。

网址：https://fossasia.org
