# 当众说话六要素

  • 声音：宏亮、清晰、抑扬顿挫
  • 语速：快慢可调
  • 目光：有交流，点视、环视、巡视、虚视
  • 表情：喜怒哀乐惊忧恐体现出来
  • 肢体语言：态势语言、动起来才美
  • 内容：条理性、概括性、思想性、讲故事

# 自我介绍万能公式

称呼问好 + 姓名 + 籍贯 + 职业 + 兴趣爱好 + 参加活动的感受 + 祝福语
