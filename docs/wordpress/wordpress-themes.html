这里收集一些我觉得挺不错的 WordPress 主题，通常都是收费的，但也有一些会同时提供基础免费版。

# 通用网站

# Astra

Astra 主题最大的卖点是它具有许多的免费样式提供使用（专业版有 70 种，免费版有 35 种）。同时，用户能根据自身的使用习惯来使用 Astra 对所创建网站的自定义布局；在使用 Astra 时，其人性化的菜单设计也能给第一次使用这款主题的用户最好的体验，比如，上传 Logo、字体样式设计等功能会直接显示在菜单上，即使是免费版本，也能满足用户建站的基本需求（专业版在这些功能上也有更好的强化升级）。

Astra 主题实例：

  • 电商主题：Furniture Store（是 Astra 众多优秀电商主题中注重于销售家具的一款主题）
  • 教育主题：Music School（是教育主题中专注于音乐教育的主题）
  • 杂志主题：Travel Agency（是杂志主题中专为旅游杂志而设计的主题）

# 资讯网站

# JustNews

JustNews 主题专为博客、自媒体、资讯类的网站设计开发，自适应兼容手机、平板设备，支持前端用户中心，可以前端发布 / 投稿文章，同时主题支持专题功能，可以添加文章专题。

# CorePress Pro

CorePress Pro 是一个强大的 WordPress 定制主题，目标是一站式解决方案，无需再安装各种各样的插件。具有体积小、性能强、功能多等特点，是一款不可多得的高性能、高颜值主题。

# 电商网站

# Sage

Sage 主题专为在线商城网站需求而开发，通过深度整合 WooCommerce 插件可让 WordPress 具备强大的电子商务网站功能，实现完整的在线购物流程。
