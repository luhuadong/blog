英语单词辨析：Competition vs. Contest

发表于 2022-06-09 更新于 2023-01-25 分类于 一起学英语
Competition 多指个人或小组之间有直接交锋，直接彼此对抗的比赛。常用于体育类的各种比赛或是知识竞赛等，比赛结果谁输谁赢比较明显，更强调参赛者之间谁比谁更加厉害。 Contest 多指一群参赛者“各自”带作品参赛或进行表演，由一组裁判判定获胜者。通常参赛者之间没有直接的交锋，例如创客作品比赛。 两者之间没有很明显的界限，经常可以互换使用。相对来说，Competition 更强调彼此竞争对抗性，Contest 则不太强调对抗性。
