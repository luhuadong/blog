# 什么是二八定律

二八定律 / 八二定律，又称为帕累托法则（Pareto principle）、80/20 法则、关键少数法则。由意大利经济学家帕累托（Vilfredo Pareto）在洛桑大学发现的。

二八定律最直观的表述是：约仅有 20% 的因素影响 80% 的结果。即在所有变因中，最重要的仅有 20%，虽然剩余的 80% 占了多数，影响的幅度却远低于 “关键的少数”。例如：世界上大约 20% 的人拥有了 80% 的财富、意大利约有 80％ 的土地由 20％ 的人口所有、80% 的豌豆产量来自 20% 的植株等等。

# 二八定律的启示

除了在经济学领域，二八定律也可以应用于其他领域，如时间管理、销售和市场营销等。

在日常工作和生活中，善用二八定律可以帮助我们取得成功。例如：

  • 重点关注关键 20% 的事情：二八定律告诉我们，我们应该把重点放在最重要的 20% 的任务、客户、收入来源等方面，以获得最大的收益。
  • 逆向引导提高工作效率：如果我们意识到自己是二八定律中的 20%，我们可以更加有效地利用自己的时间和能力，以完成更多的工作并提高工作效率。
  • 聚焦核心市场：对于企业而言，二八定律也适用于市场营销。企业应该把重点放在最有利可图的核心市场，培养忠实的客户和赢得市场领袖地位。
  • 积极管理资源：二八定律让我们认识到资源的不均衡分配，如果能够有效地管理资源，我们就可以更好地管理时间、财富和人际关系等方面。

# 二八定律的应用

二八定律表明，我们所做的工作有 80% 是浪费的，这对任何专业人士，特别是其行为对其财务收益产生直接影响的企业家和自雇专业人士而言，都是一个令人忧心的想法。不过，好消息是该法则同样可以给你带来优势。

以下是如何做到这一点的四个简单步骤：

1. 后退一步、放轻松并评估

在与工作生活拉开点距离之前，你通常不会意识到时间都花（或错误地花）在了哪里。一定要定期花一些时间后退一步，回顾一下工作，并确定哪些行动和活动产生了结果。以视觉化的方式看待事物大有帮助，例如查看来自不同供应商的统计结果饼图。拥有这种视角将有助于你根据潜在生产力确定任务的优先顺序。

2. 从小处着手

几个小步骤就可以对消除浪费的 80% 产生很大影响。首先确定经常令你白天分心的干扰因素，如电话、电子邮件、通知或临时会议等。可以着手慢慢地逐个减少，而不是一蹴而就地去除所有 “干扰因素” 来做出重大改变。

3. 学会说 “不”

学习如何说 “不” 在很大程度上可以最大化时间价值，但这有时可能极具挑战性。说 “不” 不仅适用于对人，也适用于对事，比如选择错过一个令人兴奋和创新项目，因为你已经优先考虑其他已经在管理并需要全力以赴的计划。拒绝机会可能很困难，但优先考虑更有利可图的客户或企业最终会增加产出，同时减少投入。

4. “分批处理” 任务

有些任务显然无法从你的工作日中消除，特别是如果你身边没有一个大型支持团队。“分批处理” 是一个好方法，有助于减少花在那些琐碎但必要的杂事（如清理收件箱、批准报销单或其他管理任务）上的时间。每天或每周分配出一些时间来专门处理这类事情，并且一口气全部完成，你会惊讶地发现这些工作平时消耗了你多少时间。

提高工作效率几乎不是一门精确的科学。实际上，你可能会发现，就你的情况而言，80/20 法则更接近 70/30 或 60/40 法则。无论你处于什么环境或领域，重要的是找到可以自动处理、减弱声音、拒绝（或停止）、分组或单纯说 “不” 的工作领域，以便在减少工作量的同时最大限度地提高效率。
