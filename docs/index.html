胆红素偏高意味着什么？

# 胆红素是什么 胆红素（Bilirubin）是胆色素的一种，是人类胆汁的主要色素，呈橙黄色。它是体内血红素的主要代谢产物，有毒性，可对大脑和神经系统引起不可逆的损害，但也有抗氧化剂功能，可以抑制亚油酸和磷脂的氧化。胆红素是临床上判定黄疸的重要依据，也是肝功能的重要指标。 # 胆红素检测指标 正常人血胆红素 80%-85% 来自衰老的红细胞崩解，其余来自肝内非血红蛋白的亚铁血红素和骨髓未成熟红细胞破坏。人体内胆红素共分为 “总胆红素、直接胆红素、间接胆红素”，其具有抗氧化功能以及影响着肝细胞再生，在肝细胞对胆红素的摄取、结合和排泄过程中，任何一个环节发生障碍，均可使血胆红素增高，引起黄疸。
