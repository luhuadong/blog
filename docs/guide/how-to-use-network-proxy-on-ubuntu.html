# 第 3 步：配置网络代理

点击菜单栏中的 “连接 -> 添加”，工具提供了多种添加方式，比较方便的是扫描二维码或者导入 URL 自动识别，然后再 “手动” 修改特定参数，比如本地端口号。

然后点击工具栏第一个 “连接” 图标，如无意外，现在你已经连接到网络代理了。
