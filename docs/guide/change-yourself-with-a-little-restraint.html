从一点小小的克制开始改变自己

# 生活习惯

  1. 戒掉熬夜，保证充足睡眠
  2. 起床后叠好被子，换好衣服，有仪式感地开启每一天
  3. 每天至少喝 6 杯水（早上起床后喝一大杯）
  4. 每天的早餐一定要吃（注重营养搭配）
  5. 晚饭不要吃撑，六分饱即可
  6. 每天多吃水果和蔬菜
  7. 少吃甜食和碳酸饮料
  8. 每天坚持运动 15-30 分钟
  9. 每周整理收拾一次房间，心情会变得更好
  10. 坚持理财记账的好习惯

# 自我提升

  1. 保证每个月至少读两本书
  2. 把自己的字练好
  3. 培养自己独立思考的习惯
  4. 多写随笔，提升自己的写作能力
  5. 精进学习 / 工作要用到的各种技能（如剪辑、制表、演讲、调试等等）
  6. 多对镜练习，锻炼自己的表达能力
  7. 利用碎片时间学习，看书或背单词
  8. 提升工作效率，找到适合自己的方式方法
  9. 戒掉拖延症，要做的事立马去做！

# 提升内外气质

  1. 多微笑，微笑能带来自信和好运
  2. 时刻保持抬头、挺胸、不驼背
  3. 多读书，腹有诗书气自华
  4. 穿着简洁大方，多提升衣品
  5. 多练习普通话发音
  6. 不吝啬对别人的夸奖，大方赞扬
  7. 不要悲观主义，告别负能量和抱怨

# 人际交往

  1. 减少无意义的社交，远离让你心生厌烦的人
  2. 提高圈子质量，多和优秀的人接触
  3. 不要当肥宅，多出去走走，广结善缘
  4. 记得伴侣、亲朋好友的生日并送上祝福
  5. 多和父母通电话，提供心灵上的陪伴

# 消费观念

  1. 投资自己，理性消费，不盲目跟风
  2. 养成记账的好习惯，便于每月复盘
  3. 设立目标，为自己的未来存款
  4. 学习理财知识，不懂不投（有时候减少亏损就是赚了）
  5. 省吃俭用不如尝试提高自己的收入，如发展一门副业
