开始我的环保博主生涯

发表于 2022-06-12 更新于 2022-07-30 分类于 不定时日记 Valine：

在即刻 App 发布了一周的可持续生活动态，有了珍贵的 17 位粉丝。为了更好地开始环保博主的生涯，我决定统一修改环保圈的昵称 —— 壹个兜子。为此，写了一段话，如下：

致我尊敬的17位粉丝，经过一天的深思熟虑，我还是修改昵称了！

  1. 为什么需要一个新的昵称？
    答：因为我不想和其他平台的昵称混在一起，这样可以少一些历史包袱。也可以和原来的圈子隔离开来，没有人认出我，就不用解释那么多啦。

  2. 为什么选择“壹个兜子”这个名字呢？
    答：其实我列了5个名字，但是为了保证我在其他平台（例如 B站）也能注册上，最后就只剩下这个昵称了。

  3. 是不是抄袭“壹个袋子”的？
    答：众所周知，壹个袋子是目前B站最知名的环保博主之一，这个名字在最初设想的时候确实是借鉴自“壹个袋子”。但内容不会抄袭啦，因为我们的风格完全不一样，我可能会更多地从男性视角、工程技术视角去给大家分享一些环保可持续方面的内容。另外，我也深知声称自己是一名环保主义者是很容易被人攻击的，所以多一个可以被他们攻击的对象，对其他环保博主来说也是一件好事。

  4. 为什么叫“兜子”，不叫“盒子”？
    答：我在即刻可持续生活圈子发的第一条动态就是一个饭兜（餐盒），并且引起了一些争议，这个争议其实很典型。自带餐盒也是我最早的环保行动，所以感觉还是比较有意义的！至于为什么是兜子，不是盒子，那是因为“兜子”读起来和我的本名更接近，而且我是广东人嘛，我们都管饭盒叫“饭兜”。当然啦，我也是看麦兜长大的，麦兜不仅很可爱，而且还很低碳！

  5. 还有其他名字推荐吗？
    答：如果你也想成为一名环保博主，可以考虑一下：壹个盒子、壹个瓶子、壹个杯子、壹个叉子……这些名字哈，这样我就不会感到太孤单了！

  6. 为什么想做环保博主？
    答：虽然我是一名程序员，但其实这些年来一直在关注环保，我上一份工作就是做环保设备的（现在做汽车电子），所以我感觉自己还是有一些行业经验，能够从环境、社会、经济等维度看待环保或者可持续发展。我也不喜欢搞性别对立，比如听到“都是女性在关注环保，男性都干嘛去了”、“男性的环保意识远低于女性”、“男性产生更多的垃圾”…… 我就想上去理论一番。与此同时，还有“日本人素质高，很环保，中国人就不行……”等言论。所以我想，确实应该让更多人听到环保圈里男性的声音，中国人的声音，消除大家对中国男性的刻板印象！

你们觉得“壹个兜子”这个名字怎么样？没有意见的话就这么定啦！对了，我在B站也注册了同名账号，欢迎关注哈！说不定哪天就更新了呢～

您的鼓励是我最大的动力！
