打工的目的只有一个

发表于 2023-02-01 更新于 2023-03-06 分类于 不定时日记
打工的目的只有一个，就是为自己的事业做准备，无论是积累资金、积累行业经验还是积累人脉......

整个2022年，我几乎都在加班，深度参与了公司从零到一打造的产品，为多个软件项目写下了第一行代码，也因此绩效拉满，拿下了年度优秀员工奖。

说出来你们可能不信，这是我工作八年来，第一次获得优秀员工奖。但就工作内容而言，研发部分变化并不大，还是一样写代码、写文档、做测试，相比于前两份工作经历，这些工作在创业团队中得到了放大，除个人能力以外，企业文化是其中一个很重要的原因。除此之外，我还陆续承担了 Git 仓库管家、导师、面试官等工作，积累了一些管理经验和教训，包括技术管理、项目管理、时间管理、人才培养、品牌建设。

同事们说我太卷，我说是因为不想躺平，打工就要有打工的态度！好好利用公司的平台，在这里多学点东西。如果项目成功了，跟着公司上一个台阶；如果项目不成功，也是为自己未来的事业打基础。

对我来说，打工的目的只有一个，就是为自己的事业做准备，无论是积累资金、积累行业经验还是积累人脉…… 否则，我应该趁早找份体制内的稳定工作，早点结婚生子。

可是要准备到什么时候，我们心里都没底。随着行业竞争越来越激烈、市场越来越动荡，持续的高压带来身体和心理的疲惫，似乎总是提醒我们可能随时会被击垮。每日早会变得越来越严肃，每个人都像战士一样，调动全身力气去打一场未知的仗。

我和同事B说，我们现在打工都这么拼了，是不是还不如自己去创业？如果做自己真正热爱的产品，拥有属于自己的一份事业，应该会更开心吧？

同事B脸上露出笑容，与我的想法不谋而合。他沉默了一会说，如果有好的想法，我可以当你的投资人。我说好，并在心中记下了他这句话。

我们讨厌工作，是因为工作太忙，容易会让人逐渐失去爱的能力。如果无法感受爱，工作中创造的价值也会变得黯然无光。

您的鼓励是我最大的动力！
