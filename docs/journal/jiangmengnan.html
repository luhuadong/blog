江梦南：世界以痛吻我，我却报之以歌

发表于 2022-05-25 更新于 2023-03-06 分类于 不定时日记

家里没有电视机，我也好久没上微博、知乎了，今天晚上打开了知乎查点资料，刚好看到关于江梦南的推送，好奇地看完了…… 全程都在惊叹 —— 不容易呀！然后又很开心 —— 以后会越来越好的！

江梦南，1992年出生在湖南省郴州市宜章县的一个瑶族家庭。父母为她取名的寓意是梦里江南，岁月静好。在半岁时，由于耳毒性药物导致极重度神经性耳聋，平静美好的生活被打破了。当时江梦南左耳损失大于 105 分贝，右耳听力完全丧失。失去了听力，不仅听不到外界的声音，也会听不到自己的发音，就会失去说话的能力。

下面是江梦南同学的大致经历：

  • 1992年，在湖南省郴州市宜章县出生；
  • 2007年，进入郴州市明星学校就读高中；
  • 2011年，以 615 分的成绩从郴州市明星学校考入吉林大学，就读于药学专业；
  • 2015年，在吉林大学药学院攻读硕士研究生，选择计算机辅助药物设计作为研究方向；
  • 2018年，江梦南右耳成功植入人工耳蜗，重获了失去 26 年的听力；
  • 2018年，进入清华大学生命科学学院攻读博士研究生，就读生物信息学专业，研究和免疫相关的机器学习建模课题；
  • 2020年，江梦南当选清华大学学生无障碍发展研究协会第五任会长，并荣获2021年清华大学学生年度人物；
  • 2022年1月，入选“2021感动中国之感动湖南人物”候选人名单；
  • 2022年3月3日，被评为“感动中国2021年度人物”；
  • 2022年5月4日，被公示为第19届“湖南青年五四奖章”入围个人；
  • 精彩继续……

嗯！很强对不对？感动中国年度人物实至名归！下面是央视新闻推出的 #读唇语考清华女孩原型微电影#《我的赛道》

说来也巧，我和江梦南是同年出生的，也是同年读高中，她小时候生活在湖南最南端的郴州市宜章县，我小时候生活在广东最北端的清远市。虽然我不是瑶族，不过清远有两个瑶族自治县，离宜章县很近，我也有一些瑶族的同学。

而且看她的介绍，我们现在应该还是同行 —— 程序员，在三维图像识别方面会有一些交集。于是，我在 GitHub 上盲猜她的 ID，但是没找到。。。不过，微博账号倒是很容易就找到了，内容不多，但正能量满满！果断添加特别关注！

正能量的人很多，不过我好久没有这种被深深触动的感觉了，可能是由于我和江梦南有一些共同点吧，一方面觉得她很厉害、很励志，另一方面觉得她很亲切、很熟悉。

其实我自己最近有点丧，工作反复出现倦怠，感情生活空白，也还在为过去的一些错误决策填坑。了解江梦南的事迹之后，第一感觉就是 “我这点破事算啥，好好努力就是了~”

让我印象最深刻的一句话是，江梦南说：“我从来没有因为听不见，就把自己放在一个弱者的位置上。我经常跟别人说，千万不要因为我听不见就放低对我的要求和标准。我可能取得了一点点较小的成绩，我怕他们会因为我听不见，而觉得我取得的这个成绩特别大，放大好几倍，我不太希望他们这样，我更希望他们以一个健全人的标准要求我。解决生命健康的难题，生命科学领域如果有一个难题能够被解决被攻克，那肯定造福的不只是一个人，而是造福所有人。如果我能够做一些什么让别人不再那么辛苦那么艰难，我觉得是一件很有意义的事情，如果每一个人都能够付出一点点努力，这个世界肯定是非常美好的。”

另外有一点其实感触很深，就是江梦南 26 岁时成功植入人工耳蜗重获听力之后，需要重新感受、适应、训练声音。对于从小读唇语的她来说，听别人讲话就如同学习一门外语一般，需要经历很长的语训阶段才能不再依赖唇语，凭声音去和别人交流。我也有类似的经历，由于小学和初中阶段生活都比较封闭和落后，所以我说话带有很浓的口音，记得刚去市区读高中的时候，就因为口音问题被同桌排挤。我也是从那时候开始才注意到 —— 原来听力对说话的影响那么重要！然后慢慢地去调整、训练自己的口腔和舌头，才学会了比较正常的普通话和粤语。难度真的不亚于重新学一门外语！因此让我更加佩服江梦南同学，这一路走来真的很不容易！

幸运的是，江梦南终于走到了这里，感恩她的父母耐心教导，感恩她的不懈努力！越努力，越幸运！祝福江梦南同学，期待未来有一天能认识你呀！

您的鼓励是我最大的动力！
